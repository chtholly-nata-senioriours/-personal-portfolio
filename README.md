# Personal Portfolio

个人作品集网站，展示项目经验、技术能力与博客文章。

## 技术栈

- **框架**: Next.js 16 (App Router)
- **前端**: React 19, TypeScript 5
- **样式**: Tailwind CSS 4
- **动效**: Framer Motion
- **博客**: Markdown / MDX

## 快速开始

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看。

## 项目结构

```
src/
├── app/            # 路由页面
├── components/     # UI 组件
├── content/blog/   # 博客文章（Markdown）
└── lib/            # 工具函数
```

## 博客

在 `src/content/blog/` 下新建 `.mdx` 文件即可自动添加文章，无需修改代码。
